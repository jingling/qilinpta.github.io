What is QiLin?

QiLin is a new pointer analysis framework for Java programs. QiLin supports fine-grained context-sensitivity ( where different variables/objects in a method can be analyzed under different context abstractions at the variable level), precisely, efficiently, and modularly.

What kind of analyses does QiLin provide?

  • Precise Call graph construction for Java programs
  • Andersen's pointer analysis
  • All kinds of Method-level context-sensitive pointer analyses, e.g., kCFA, kOBJ, kType, kHybrid
  • Existing selective context-sensitive pointer analyses, e.g., Bean, Zipper, Mahjong, Eagle, Turner, Context-Tunneling, Data-Driven.

How to run QiLin?

QiLin analyzes a Java program by taking its class files or Jar files as input. Please refer to this user guide to run QiLin with a simple example and generate the analysis outputs in your local machine.

Alternatively, we have also prepared a docker image for you to run QiLin on any machine with Docker installed. Please refer to this step-by-step guide to try QiLin in a Docker container.
